# **Dr MED**

Be sure about your medicines.

## **Features**

1. Chat with medical expert
2. Report adverse drug reaction
3. Health Tips
4. Health Tip for elderly people
5. Webinar & Health Information videos

> ### App supports languages *English* & *Malayalam* only
